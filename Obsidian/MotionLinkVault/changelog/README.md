# Changelog

Parent: [[index]]

Каждый файл здесь — `YYYY-MM-DD.md`, содержащий записи изменений за день.

Формат записи:

```markdown
## [filename or module] — [краткое описание изменения]
- Что изменилось: ...
- Почему: ...
- Связанные файлы: [[modules/...]]
```

Создавать запись только для **значимых** изменений (новый модуль, изменение поведения, удаление API, изменение протокола). Мелкие правки UI / опечатки сюда не идут.
